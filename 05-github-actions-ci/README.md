# Project: CI/CD with GitHub Actions

## Description
Automate your development workflow! This project sets up a GitHub Actions pipeline to automatically run linting and unit tests whenever code is pushed to the repository.

## Objectives
- Understand the structure of a YAML workflow file.
- Set up a build job in GitHub Actions.
- Trigger jobs on `push` and `pull_request` events.

## Prerequisites
- GitHub repository

## How to Run
1. Copy the `.github/workflows/ci.yml` file to your own repo.
2. Push a change to the `main` branch.
3. Check the "Actions" tab in your GitHub repository to see the pipeline running.
